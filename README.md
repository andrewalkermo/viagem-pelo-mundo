## SENTIDO
### Formato
Letra "O" para Oeste ou "L" para Leste
### Exemplo
O
## LONGITUDE_ORIGEM
### Formato
decimal, seguido da letra "O" para Oeste ou "L" para Leste
### Exemplo
32L

## LONGITUDE_DESTINO
### Formato
decimal, seguido da letra "O" para Oeste ou "L" para Leste
### Exemplo
59O

## DURACAO_VIAGEM
### Formato
time (HH:ii:ss)
### Exemplo
15:32:59

## DATETIME
### Formato
datetime (yyyy-mm-dd HH:ii:ss)
### Exemplo
2021-10-27 22:01:52
